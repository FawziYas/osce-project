# Official Letterhead Background

Place your university's official letterhead background image here with the name: **letterhead-background.png**

This background will appear on all printed score sheets, giving them the formal appearance of official university documents.

**Recommended specifications:**
- Format: PNG (with transparent background preferred)
- Size: 210mm × 297mm (A4 size) at 300 DPI = 2480 × 3508 pixels
- Or at least: 1654 × 2339 pixels (200 DPI)
- File name: **letterhead-background.png**

**Design Tips:**
- Include university logo, name (Arabic & English), and any official headers at the top
- Keep the center area clear for document content
- Use light colors or watermarks to avoid interfering with printed text
- Test print before final use to ensure readability

**Example layout:**
```
┌─────────────────────────────────────┐
│  [Logo]  جامعة النجاح الوطنية      │ ← Header area (120px)
│     An-Najah National University    │
├─────────────────────────────────────┤
│                                     │
│     Content area (student scores)   │
│                                     │
│                                     │
└─────────────────────────────────────┘
```

If the file is not found, score sheets will print with a plain white background.
